# M.E Hydraulics – Business Website

A professional single-page business website built for **M.E Hydraulics**, a hydraulic services company based in Njiru, Nairobi, Kenya.

## 🔧 About the Project

M.E Hydraulics offers hydraulic hose repairs, pipe pressing, hydraulic oils, spare parts, and excavator hire services. This website was designed to give the business a professional online presence and make it easy for clients to get in touch.

## ✨ Features

- Responsive single-page layout optimised for mobile and desktop
- Sticky navigation bar for easy section access
- WhatsApp deep links for instant client contact
- SEO optimisation with structured data (JSON-LD) and Open Graph meta tags
- Accessibility improvements for wider reach
- Deployed live via Firebase Hosting

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure and semantic markup |
| CSS3 | Styling, responsive layout |
| JavaScript | Interactive elements, navigation |
| JSON-LD | Structured data for search engines |
| Firebase Hosting | Deployment and hosting |

## 🚀 Getting Started

No build tools required. Simply open `index.html` in a browser, or deploy via Firebase Hosting.

```bash
git clone https://github.com/YOUR_USERNAME/me-hydraulics-website.git
cd me-hydraulics-website
# Open index.html in your browser
```
